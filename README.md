# Instruction

1.  Clone Project

```
 git clone https://github.com/PhurinatBunyachai/dev-deva-frontend-test.git
```

2.  Install Dependencies

```
 npm install
```

3.  Create .env file

```
 echo "APP_BACKEND_URL = 'http://localhost:3000/'"\\n"APP_BASE_URL = '/'" > .env
```

4.  Run Project

```
 npm run dev
 or
 pnpm run dev --port 4000
```
